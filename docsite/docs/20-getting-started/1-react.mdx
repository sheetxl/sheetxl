## New Project

The fastest way to get started with SheetXL is using our project wizard. It will create a new
project with SheetXL pre-configured and ready to use.

## Existing Project

If you want to add SheetXL to an existing React project, follow these installation steps.

### Installation

To start install the following packages:

`@sheetxl/studio-mui`

And additional peer dependencies unless you already have these in your project.

* `react`, `react-dom` - React
* `@emotion/react`, `@emotion/styled` - CSS-in-JSS Styling (Material UI dependency)
* `@mui/material`, `@mui/styled-engine`, `@mui/system`, `@mui/utils` - Material UI

<Tabs
  defaultValue="pnpm"

  </TabItem>
</Tabs>

Once installed SheetXL components can be used as any React component.

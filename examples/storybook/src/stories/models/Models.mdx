# Overview

SheetXL is a model View control design that decouples the view (widgets) from the data (models).

## Models

SheetXL is implemented as a headless TypeScript library that can be used in the browser or in a NodeJS environment.
See the [Guide](https://www.sheetxl.com/docs/guides/concepts/concept-overview) or the [API](https://api.sheetxl.com/).
